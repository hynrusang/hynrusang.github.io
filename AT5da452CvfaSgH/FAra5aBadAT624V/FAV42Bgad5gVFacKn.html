Html의 기본

    <DOCTYPE html>
  • 이 문서가 html 문서라는 것을 지정해줍니다.
    <html> </html>
  • html 문서의 내용을 사이에 입력합니다.
    <head> </head>
  • 문서의 header 부분을 지정해줍니다. 주로 <meta charset>이나 <title>이 들어갑니다.
    <meta charset="인코딩할 방식">
  • 문서를 어느 방식으로 인코딩 할지 지정해줍니다. 예를 들어 이 문서는 <meta charset="UTF-8">로 되어 있으며, 이것은 이 문서가 UTF-8 방식으로 인코딩되었음을 의미합니다.
    <title> </title>
  • 문서의 최상단에 올라가는 제목을 지정해줍니다.
    <body> </body>
  • 문서의 본문을 지정해줍니다.
    <h> </h>
  • 글을 출력해줍니다. 총 5단계가 있습니다.
      • <h1> </h1>
    • 이런 크기로 글을 출력해줍니다.
      <h2> </h2>
    • 이런 크기로 글을 출력해줍니다.
      <h3> </h3>
    • 이런 크기로 글을 출력해줍니다.
      <h4> </h4>
    • 이런 크기로 글을 출력해줍니다.
      <h5> </h5>
    • 이런 크기로 글을 출력해줍니다.
    <p> </p>
  • 문서의 단락을 지정해줍니다. 주로 하이퍼링크나 주석을 달 때 많이 사용합니다.
    <pre> </pre>
  • 문서의 장락을 지정해줍니다. (<p> </p>)와 다르게, (<pre> </pre>)내에선 모든 공백, 뛰어쓰기를 "모두" 인식합니다.
    <hr>
  • 이런 절취선을 출력합니다.
    <br>
  • 뛰어쓰기를 출력합니다. 만약 여기에 <br>를 넣는다면
    이렇게 됩니다.
    style
  • html 문서를 쓰는 이유기도 합니다. (<style> </style>)이나 <Html 태그명 style="속성명 : 값;">으로, 방식이 2개로 나뉩니다.
      • <style> </style>
    • 문서의 html 태그 전체를 아우르는 style을 지정할 수 있습니다. 그에 대한 설명은 나중으로 미루겠습니다. 주로 <body> 내에서 첫번째로 지정합니다.
      <Html 태그명 style="속성명 : 값;">
    • 해당 Html 태그에"만" 적용되는 style을 지정할 수 있습니다. 그에 대한 설명은 나중으로 미루겠습니다. 주로 글자 색이나, 크기, 배경색을 바꾸는데 쓰입니다.
    <script>
  • html 내에서 javascript를 출력합니다. 그에 대한 설명은 나중으로 미루겠습니다. 주로 <body> 내에서 두번째로 지정합니다.
