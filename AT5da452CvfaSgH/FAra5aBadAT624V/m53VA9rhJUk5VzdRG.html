  • 문서의 html 태그 전체를 아우르는 style을 지정할 수 있습니다. html 태그:커멘드 { }로 구성됩니다.
      • html 태그 { }
    • 평상시 해당 html 태그에 이하 수식언이 작동합니다. 아래는 그 사용 예시입니다. 이 문서에도 적용되어 있으니 확인해보세요.
      • h3, li, a {
	transition : all 2s;
}
      html 태그:hover { }
    • 해당 html 태그 위에 마우스를 올려놓을 때 작동합니다. :link :visited :active 등이 덮어 씌울 수 있기에,
      하이퍼 링크 스타일에는 :link-:visited-:hover-:active 순으로 정의됩니다. 아래는 그 사용 예시입니다. 이 문서에도 적용되어 있으니 확인해보세요.
      • h3:hover, a:hover {
	color : cyan;
	background : turquoise;
}
      html 태그:link { }
    • 해당 링크가 아직 방문하지 않았거나, :link 이외에 다른 커멘드가 없을 시에 평상시 작동합니다. :visited는 그 반대로 작동합니다.
      보통 html 태그에 a를 지정합니다. 아래는 그 사용 예시입니다. 이 문서에도 적용되어 있으니 확인해보세요.
      • a:link {
	color : darkcyan;
}
      html 태그:active { }
    • 사용자가 해당 html 태그를 활성화했을 때 작동합니다. 예를 들어, 마우스인 경우
      마우스로 해당 html 태그를 클릭하고 때는 순간까지를 의미합니다. 아래는 그 사용 예시입니다. 이 문서에도 적용되어 있으니 확인해보세요.
      • li:active {
	font-size : 160%;
}
      html 태그[수식언=상태]:커멘드 { }
    • html 태그가 해당 수식언과, 해당 상태를 갖고 있을 시, 작동합니다. 아래는 그 사용 예시입니다. 이 문서에도 적용되어 있으니 확인해보세요.
      • input[type=text]:hover {
	color : cyan;
	background : orange;
}
input[type=password]:hover {
	color : white;
	background : black;
}
