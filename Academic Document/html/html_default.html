Html에 대하여 (기본편)

    Html이란?
  • Html은 거의 모든 웹 페이지를 만들어주는 파일입니다.
    그냥 메모장 같이 "only 글"로만 적힌 파일이라고 생각하시면 됩니다.
    Html은 그 자체로 단독으로 쓰이기도 하지만,
    Html을 꾸며주는 Css 파일이나, Html에 두뇌를 만들어주는 js 파일과 같이 쓰이는게 일반적입니다.
자 그럼 이번엔 실제로 Html 파일을 만들어 볼까요?
보통은 Html, Css, js로 구성하는 편이 일반적이지만, 
이번에는 이해를 위해, Html만 이용해서 웹 페이지를 만들어봅시다.
